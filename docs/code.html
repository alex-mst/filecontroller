Code Documentation¶

[!] Things to keep in mind¶

  • Please use the name of the object we will set up to run any commands, not the name of the class.

  • Any functions that have to do with modifying files with numbers start from 1 instead of 0, to make FileController more user-friendly. So, if you wanted to do fc.deleteLine('text_file.txt', '0') to delete the first line, you’d type ‘1’ instead of ‘0’.

  • If a function does not work, please get in contact with me with the contacts I’ll be leaving below and I will do my best to fix it on the next release.

[I] Getting Started¶

To start using FileController, we will need to do a few things. Here is the basic configuration for the package: 1. Import the module by typing ‘from filecontroller import ‘ 2. Make a object and assign it to the ‘fileController()’ class, like shown: ‘your_object = fileController()’ You will use that object every time you want to call a function from FileController 3. You can type any function by using ‘your_object.function()’ **NOTE:* from now on we will be refering to the object as ‘fc’.

[II] Functions¶

Functions in FileController can do all the basic file management command in Python and there are also some more advanced tools included like a ZIP manager etc. Here is a quick table with the functions and a small usage:

Usage

fc.writeToFile(filename, text)

Writes text to the file

fc.editFile(filename, line, text)

Edits a specific line of a file

Deletes an entire file

fc.deleteLine(filename, line)

[III] Usages¶

After you saw the table above, you may have a general idea of what each function does. Below there are the exact usages along with other information that may be useful:

fc.readfile(filename): Reads each line of a specific file given by the user and prints it back. In this command .txt is required

fc.writeToFile(filename, text): Writes text given by the user to the bottom of the file. In this command, .txt is required

fc.editFile(filename, line, text): Edits a specific line of a file with the text given.

fc.deleteFile(filename): Deletes all the contents of a specific file

fc.deleteLine: Deletes a specific line for a file. As we mentioned in the beginning, keep in mind that this command does not work with the binary system, so position 1 on the list is ‘1’ and not ‘0’, along with the others.

fc.getList(filename, listname): Makes a list with the lines of a file. This function must be assigned to a variable that can be later used as a normal list

fc.writeList(filename, listname): Writes a pre-existing list on a specific file. This command, along with the other, can be used if you want to manually edit the contents of a file and add them back to the file instead of using the .editFile() function

fc.createFile(filename): Creates a file with a name given by the user into the same directory

fc.zipFiles(number, zipname): Adds files into a ZIP file (currently works only with text files, will be updated on newer versions). Can be used with or without .txt at the end of the file, as well as with different directories.

fc.renameFile(old_name, new_name): Changes the name of a file

fc.readZIP(filename): Reads the contents of the ZIP file and gives a tree view of the directories to the console. (can also work with different directories)

fc.combine(file1, file2, merged_file): Combines 2 text files into 1 (Do not put .txt at the end of the merged file)
